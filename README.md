# Coursera E-Learning Platform 🚀

[cite_start]An educational platform inspired by Coursera, designed to provide users with a seamless and interactive learning experience[cite: 13, 14]. [cite_start]Developed as a Web Programming project at Arab International University (AIU)[cite: 6, 12].

## 📝 Project Description
[cite_start]The platform allows users to register accounts, browse a wide variety of online courses, view the latest news and promotional offers, and access a Frequently Asked Questions (FAQ) section for guidance[cite: 15]. [cite_start]It is intended to enhance digital learning accessibility, streamline course management, and provide a user-friendly interface for learners of all ages[cite: 16].

## ✨ Key Features
* [cite_start]**User Authentication:** Secure login and registration system with data validation to ensure only authorized users can access the platform[cite: 17, 18].
* [cite_start]**Dynamic Home Page:** Displays news, offers, daily tips, and an image slider to engage users and highlight important updates[cite: 17, 19].
* [cite_start]**Course Management:** Comprehensive courses page where users can explore all available courses, including course descriptions, schedules, and instructors[cite: 17, 20].
* **Admin Management Panel:** Full dashboard built for administrators to seamlessly add, modify, and manage courses and prices in real-time.
* [cite_start]**FAQ Section:** Provides answers to common questions to assist users in navigating the platform efficiently[cite: 17, 21].
* [cite_start]**Responsive Design:** Fully responsive layout using HTML, CSS (Flexbox, Media Queries), and JavaScript, ensuring compatibility across different devices and screen sizes[cite: 17, 22].

## 🛠️ Technologies Used
* [cite_start]**Frontend:** HTML5, CSS3 (Flexbox & Media Queries), JavaScript (ES6)[cite: 24, 25, 28].
* [cite_start]**Backend:** PHP[cite: 24, 26].
* [cite_start]**Database:** MySQL[cite: 24, 27].

## 🚀 How to Run the Project Locally
1. Clone this repository to your local machine.
2. Move the project folder into your local server directory (e.g., `C:/xampp/htdocs/`).
3. Open **XAMPP Control Panel** and start **Apache** and **MySQL**.
4. Go to `http://localhost/phpmyadmin/`, create a new database named `user_db`, and import the provided `user_db.sql` file.
5. Open your browser and navigate to `http://localhost/coursera_project/login.php`.
